# 🖼️ Museum Artwork Analysis using SQL

## 📌 Project Overview

This project focuses on analyzing a museum artwork dataset using SQL to uncover meaningful insights related to:

- Museum collections
- Artist popularity
- Painting styles
- Artwork pricing
- Museum locations
- Data quality issues
- Business opportunities

The objective of this project is not only to write SQL queries but also to transform raw data into actionable business insights that can help improve museum operations, visitor engagement, and revenue strategies.

---

# 🛠️ Tools & Technologies

- SQL Server
- SQL Queries
- Data Cleaning
- Exploratory Data Analysis
- Business Analysis

---

# 🎯 Business Questions Explored

The analysis answers questions such as:

- Which artworks are not displayed in museums?
- Which museums have the largest collections?
- Which artists have the highest number of artworks?
- Which painting styles are most popular?
- Which artworks have premium pricing?
- Which locations have the highest museum presence?
- How can museums improve engagement and revenue?

---

# 🧹 Data Cleaning & Quality Analysis

Before analysis, multiple data quality issues were identified:

## Duplicate Records

Duplicate records can create:

- Incorrect artwork counts
- Inflated analysis results
- Poor data reliability

**Solution:**

Removed duplicate records and improved data consistency.

---

## Invalid Museum Information

Identified:

- Invalid city values
- Incorrect museum hour entries

**Impact:**

Incorrect location and operational data can affect reporting and decision-making.

**Recommendation:**

Implement validation rules and regular data quality checks.

# 💰 Artwork Pricing Analysis

## Premium Artwork Analysis

Identified artworks where:

**Asking Price > Regular Price**

### Possible Reasons:

- Artist reputation
- Rare artwork
- Premium pricing strategy

### Recommendation:

Analyze demand trends and optimize artwork valuation.

## Discounted Artwork Analysis

Identified artworks where:

**Asking Price < 50% of Regular Price**

- Incorrect valuation
- Changing market trends

### Recommendation:

Review pricing strategy and promotional opportunities.
